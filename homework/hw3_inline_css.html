Programming in Java

There are many programming languages out there to choose from. When I decided to give programming a shot it came down to choose between two languages: Java and C++. After talking it over with a few of my friends who are currently working as programmers I have decided that I should start with Java.

Starting out programming from scratch was quite a challenge. However, I believe that Java is a langauge that is easy to pick up as a novice programmer. Once you have setup your coding environment all you need is to have a good guide to follow. I think it is essencial to have a good book when you start programming.

The book that really worked for me was Starting out with Java: From Control Structures through Objects 4th Ed. by Tony Gaddis.You can find a link to the book here. This book has a way of clearly explaining how to program in Java. It is also able to provide good and clear examples of code that the user is able to follow and understand. As such, I strongly recommend this book to anyone who wants to give programming in Java a try. Here is what the book looks like.
